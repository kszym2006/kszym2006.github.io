宝大游戏模式V2服务端配置方法

发表于 2016-04-15   |   分类于 游戏模式   |  

下载

可以通过一下途径获得
服务器X64 链接: 链接: http://pan.baidu.com/s/1o6VIMKU 密码: cbsu
服务器X86（32位） 链接: http://pan.baidu.com/s/1gevFsxd 密码: g3vj
wget 直接用最新的 http://xqc.co:20001/123/game-server

配置

  • game-server放置在你喜欢的目录下（本文例：/etc/game）
  • chmod +X 766 game-server#赋予game-server 766权限
  • 创建json文件（本文例：1.json）
  • 1.json写入内容（本文例多端口版本）
    {
    “server”:”0.0.0.0”,
    “local_address”: “127.0.0.1”,
    “local_port”:1080,
    “port_password”: {
    “XXXX”: “XXXX”,
    “XXXX”: “XXXX”,
    “XXXX”: “XXXX”
    },
    “timeout”:600,
    “method”:”chacha20”,
    “fast_open”: false
    }

运行

  • 先cd /etc/game
  • 然后使用nohup ./game-server -w koolshare.github.io -c 1.json > z.log 2>&1 &
  • 已经可以食用了

开机自启动

  • 先在 /etc/game目录下新建一个s.sh
  • 然后写入
    cd /etc/game
    nohup ./game-server -w koolshare.github.io -c 1.json > z.log 2>&1 &
  • 赋予s.sh 766权限
  • 然后在etc目录下找到rc.local
  • 在exit 0之前写入
    bash /etc/game/s.sh
